# **App Name**: MooManager

## Core Features:

- Splash Screen: Display a splash screen when the app launches.
- Language Toggle: Implement language toggle for English and Urdu.
- RTL Layout: Change app layout to Right-to-Left (RTL) when Urdu is selected.
- Local Storage: Locally store the language preference.
- Dashboard: Home screen displaying essential information and quick actions.

## Style Guidelines:

- Primary color: A calming, earthy green (#70A964) to reflect nature and growth.
- Background color: A very light off-white (#F2F0EC) for a clean, accessible interface.
- Accent color: A warm orange (#E27D60) for calls to action and highlights, providing contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- When Urdu is selected, use a suitable Nastaliq font.
- Implement a consistent and intuitive layout, switching to RTL when Urdu is selected.
